== Description ==

Hides the wordpress generator meta tag from the HEAD section of your blog view source.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the folder `hide-generator-meta-tag` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What does it do? =

It will hide the meta tag where it says on which wordpress version you're working.

= Why would i want to use it? =

In some cases it can help preventing people with bad intentions to know on which wordpress
version you are currently running.
